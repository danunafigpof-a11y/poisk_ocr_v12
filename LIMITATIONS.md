# LIMITATIONS — Известные ограничения POISK OCR v12

## v12 (текущая)

### Технические
- ⚠️ **Иконки PWA не предоставлены** — `icons/icon-192.png` и `icons/icon-512.png` нужно создать вручную или сгенерировать
- ⚠️ **Только российские номера** — другие форматы не тестировались
- ⚠️ **Камера требует HTTPS или localhost** — иначе `getUserMedia` не работает
- ⚠️ **Нет автотестов** — только ручное тестирование
- ⚠️ **CDN-зависимости** — если CDN недоступен, оффлайн-режим не спасёт при первой загрузке
- ⚠️ **iOS Safari** — частичные ограничения PWA (нет полной поддержки)

### Производительность
- 🐌 Первая загрузка OpenCV.js (~8 МБ) — медленная
- 🐌 Tesseract.js инициализируется 2-5 сек
- 🐌 На слабых устройствах — возможны лаги при обработке

### UX
- 📱 Адаптация под планшеты не идеальна
- 🌙 Тёмная тема может иметь контрастные баги
- 🔔 Звуковые уведомления работают только после пользовательского жеста

### Безопасность
- 🔒 Все данные хранятся локально (нет сервера)
- 🔒 Никакой телеметрии нет
- 🔒 Камера используется только по запросу пользователя

## Известные баги
- На Samsung Internet не работает вибрация при успешном распознавании
- При первом запуске на iOS 16+ требуется двойное подтверждение камеры
- На маленьких экранах (<360px) может обрезаться кнопка "Сканировать"
